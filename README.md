# 🏅 Explorador do Prêmio Nobel

O Explorador do Prêmio Nobel é uma aplicação web que permite pesquisar, filtrar e explorar os vencedores do Prêmio Nobel ao longo dos anos, em todas as categorias.
O objetivo é oferecer uma ferramenta interativa e de fácil uso para estudantes, pesquisadores e curiosos que desejam conhecer os laureados e suas contribuições.

# ✨ Sobre o Projeto

A aplicação apresenta uma interface moderna e leve, com busca por nome, filtro por ano e filtro por categoria, exibindo os resultados de forma dinâmica e organizada em cartões.
Os dados são obtidos de uma API pública do Prêmio Nobel, garantindo informações atualizadas e confiáveis.

# 🎯 Objetivos

✅ Exibir os vencedores do Prêmio Nobel com seus respectivos anos, categorias e nomes.

✅ Permitir pesquisa por nome do laureado.

✅ Oferecer filtros por categoria e ano para facilitar a exploração dos dados.

✅ Garantir uma interface amigável e responsiva, com carregamento rápido e visual agradável.

✅ Proporcionar uma experiência fluida e intuitiva ao usuário.

# 🛠️ Tecnologias Utilizadas

HTML → Estrutura e organização dos componentes da página.

CSS → Estilização moderna com foco em simplicidade e responsividade.

JavaScript → Manipulação dos dados da API, filtros dinâmicos e renderização dos resultados.

API Nobel Prize → Fonte dos dados sobre os ganhadores de cada ano e categoria.

# 📚 Funcionalidades Principais

✅ Pesquisa por Nome: Encontre rapidamente laureados digitando parte ou o nome completo.

✅ Filtro por Categoria: Selecione áreas como Física, Química, Medicina, Literatura, Paz ou Economia.

✅ Filtro por Ano: Escolha um ano específico para visualizar apenas os vencedores daquele período.

✅ Resultados Dinâmicos: Exibição automática conforme os filtros aplicados.

✅ Interface Responsiva: Totalmente adaptada para uso em computadores, tablets e smartphones.

# 🎨 Diferenciais do Projeto

✨ Design Elegante: Paleta de cores suaves e tipografia clássica inspirada no estilo dos prêmios.

✨ Usabilidade: Busca e filtros otimizados para interação rápida e intuitiva.

✨ Código Limpo: Estrutura modular e de fácil manutenção.

✨ Atualização Automática: Os dados são carregados diretamente da API, sem necessidade de atualização manual.
